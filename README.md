Soteria QT Wallet Bootstrap Scripts

These scripts help you bootstrap the Soteria QT wallet quickly — no need to resync the entire blockchain manually.

Supported Platforms

    Windows

    macOS (CLI and GUI)

    Linux

macOS Instructions

You have two options:
1. Terminal (CLI)

Use Soteria-Bootstrap-macOS.sh:

chmod +x soteria-bootstrap-macos.sh
./soteria-bootstrap-macos.sh

2. Double-click (GUI)

Use BootstrapMacOS.command — just double-click to run.
⚠️ Important Notes

    These scripts assume your wallet is installed in the default data directory.

    If your setup is custom, edit the script paths to match your Soteria installation.
